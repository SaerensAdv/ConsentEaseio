## Usage

Each agent markdown file contains:
- **Role:** What the agent does
- **Core Responsibilities:** Key tasks
- **Frameworks/Templates:** Useful structures
- **Collaboration:** How agents work together

## 2025 Features

### Replit Agent 3 Integration
- **Fast Mode:** 2-3 seconds faster per action
- **Max Autonomy:** 200 minutes continuous operation
- **Self-testing:** Automatic browser testing
- **Checkpoints:** Rollback code, chat, and database

### Model Context Protocol (MCP)
- Universal tool connection standard
- 100+ pre-built servers (GitHub, Slack, Postgres)
- Linux Foundation governance
- SDKs for all major languages

### Multi-Agent Patterns
- CrewAI for team collaboration
- LangGraph for stateful workflows
- ReAct reasoning (Think → Act → Observe)

## Replit Optimized

- React + Vite frontend
